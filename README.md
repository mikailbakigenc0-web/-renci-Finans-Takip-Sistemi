# -renci-Finans-Takip-Sistemi
Öğrenci Finans Takip Sistemi; Python tabanlı, kullanıcı dostu bir bütçe yönetim aracıdır. Harcamalarınızı kategorize ederek (Gıda, Eğitim, Ulaşım vb.) anlık kayıt altına almanızı ve toplam maliyetlerinizi analiz etmenizi sağlar. JSON formatında veri saklama özelliği sayesinde kayıtlarınız güvenle korunur.
